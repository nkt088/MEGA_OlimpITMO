# MEGA_OlimpITMO

Для того чтобы запустить проект на вашем MacBook:
1. Перейдите в AppStore
2. Установите XCode (10-20GB)
3. Запустите XCode
4. Программа при запуске предложит вам установить дополнительные пакеты
5. Установите iOS (обязательно) и MacOS
6. Нажмите Open an Excisting Project
7. Откройте папку проекта и выберите файл с именем проекта и расширением .cxproject
	В проекте отсутсвует API ключ для sstats
	Откройте файл NetworkLayer->Sstats
	Вставьте ключ в нужно поле
8. Выберите девайс для тестирования, например iPhone 17 Pro
9. Нажмите кнопку Play - начнет сборка проекта
10. Проект запущен, реализован минимальный функционал, поиск выполняется через фильтры.
